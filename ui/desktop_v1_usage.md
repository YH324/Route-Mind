# RouteMind 桌面 Web V1 操作指引

## 打开方式

直接在浏览器打开：

`ui/route_planner_desktop_v1.html`

推荐视口：

`1440 x 900`

## 可交互项

- 点击 `中文 / EN` 切换界面语言。
- 点击右侧三张方案卡切换路线方案。
- 点击 `复制路线` 会出现复制成功 toast。
- 地图上的 `+ / - / 归中` 和图层按钮为第一版视觉占位，用于 Figma 和前端实现说明。

## 设计定位

这版聚焦桌面 Web。整体是城市工具感，加入暖橙色行动按钮和本地生活氛围，适合产品评审、方案确认和前端实现对齐。
