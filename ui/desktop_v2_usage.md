# RouteMind 桌面 Web V2 操作指引

## 打开方式

直接打开：

`ui/route_planner_desktop_v2.html`

推荐视口：

`1440 x 900`

## 可交互项

- 点击右侧方案卡切换：紧凑高效、休闲慢游、美食探店。
- 点击 `复制` 显示复制成功 toast。
- 点击 `跳过` 可隐藏新手指引。
- 点击 `清爽 / 暖橙 / 夜游` 可切换简单皮肤。
- 地图上的 `+ / - / 归中` 和图层按钮是视觉占位，用于后续 Figma 与前端说明。

## V2 改进点

- 地图从粗网格升级为插画地图，加入道路、楼宇、公园、树木和花草。
- POI 节点使用类型图标：景点、火锅、茶馆、夜逛。
- 文案更短，主要按钮更集中，降低初次使用压力。
- 增加可跳过的新手指引。
- 增加简单皮肤选择，满足不同用户偏好和展示场景。
